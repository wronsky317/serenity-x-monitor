# Long-Term Views

This directory maintains the durable thesis layer for Serenity / `@aleabitoreddit`.

It is separate from runtime artifacts:

- `raw/`: original Supercycle/X captures.
- `parsed/`: complete parsed archives.
- `reports/`: daily/report-ready summaries.
- `long_term_views/`: maintained long-term investment themes, core asset maps,
  thesis changes, and watchpoints.

## Maintenance Rules

1. Only update this directory after reading the latest local `parsed/` or
   `reports/` outputs.
2. Do not paste raw full tweet dumps here. Summarize durable views only.
3. Record the source run or date range used for each major update.
4. Keep uncertainty explicit. Distinguish:
   - Serenity's stated/high-conviction views
   - inferred portfolio/theme implications
   - watch-only or negative views
5. Track invalidation conditions, not only upside catalysts.

## Current Files

- `serenity_core_asset_map.md`: core themes, companies, cycles, watchpoints,
  and long-term logic factors derived from the 2026 backfill.
- `maintenance_rules.md`: rules for when and how future agents should update
  the long-term thesis layer.
- `pending_updates/`: unreviewed daily candidate updates waiting for manual
  merge.
- `merged/`: reviewed pending updates that have already been merged into the
  maintained core asset map.
