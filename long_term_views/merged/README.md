# Merged Long-Term Updates

This directory archives daily pending updates after their durable thesis content
has been manually merged into `long_term_views/serenity_core_asset_map.md`.

Use `git mv` from `long_term_views/pending_updates/` into this directory after
merge review. Keeping merged files here preserves the audit trail while leaving
`pending_updates/` focused on unreviewed candidates.
